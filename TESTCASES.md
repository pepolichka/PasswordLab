# TESTCASES — PASSWORD LAB

Ниже приведены тест-кейсы для ручной проверки после открытия `index.html` через `file:///`.

1. Базовая генерация
- Шаги: открыть `index.html`, нажать `Сгенерировать`.
- Ожидание: появляется пароль длиной 16, сообщение об успешной генерации, запись добавляется в историю.

2. Минимальная длина
- Шаги: установить `Длина = 6`, нажать `Сгенерировать`.
- Ожидание: пароль длиной 6 создаётся без ошибок.

3. Максимальная длина
- Шаги: установить `Длина = 64`, нажать `Сгенерировать`.
- Ожидание: пароль длиной 64 создаётся без ошибок.

4. Нет наборов символов
- Шаги: снять чекбоксы строчных/заглавных/цифр/символов, нажать `Сгенерировать`.
- Ожидание: ошибка `Выберите хотя бы один набор символов.`.

5. Исключение похожих символов
- Шаги: оставить только буквы и цифры, включить `исключить похожие`, сгенерировать 20+ раз.
- Ожидание: отсутствуют `O`, `0`, `I`, `l`, `1`.

6. Без повторов (валидный случай)
- Шаги: включить `без повторов`, длина 12, включить все наборы, нажать `Сгенерировать`.
- Ожидание: символы в пароле не повторяются.

7. Без повторов (ошибка длины)
- Шаги: включить только `цифры`, включить `без повторов`, длина 12, нажать `Сгенерировать`.
- Ожидание: ошибка о том, что длина больше количества уникальных символов.

8. Копирование
- Шаги: сгенерировать пароль, нажать `Скопировать`.
- Ожидание: появляется `Скопировано!`; при ограничении браузера показывается понятное сообщение про ручное копирование.

9. Проверка слабого пароля
- Шаги: ввести `password123`, нажать `Проверить`.
- Ожидание: статус `Слабый`, низкий балл, предупреждение о популярном пароле и советы.

10. Проверка сильного пароля
- Шаги: ввести `vT$8mQ!2zR#9Lx@4`, нажать `Проверить`.
- Ожидание: статус `Надёжный`, высокий балл, большинство пунктов с `[OK]`.

11. Проверка последовательностей
- Шаги: ввести `Qwe123!!AA`, нажать `Проверить`.
- Ожидание: балл снижается из-за последовательностей `qwe` и `123`.

12. История и очистка
- Шаги: сгенерировать 12 паролей подряд, затем нажать `Очистить историю`.
- Ожидание: хранится только 10 последних записей; после очистки отображается сообщение, что история пуста.
